# MindBreath - Mental Wellness App

A mental wellness application featuring guided breathing exercises, interactive games, and progress tracking to help users reduce stress and improve focus.

## Features

### Breathing Exercises
- Four breathing modes: Calm, Focus, Deep Relax, and Sleep
- Visual breathing guide with animated circle
- Real-time statistics tracking cycles and duration
- Session history saved locally and to cloud

### Interactive Games
- Aim Trainer - Test reflexes with increasing difficulty
- Memory Match - Boost cognitive function
- Focus Dot - Improve concentration
- Additional wellness games included

### Dashboard
- Mental state analysis with stress, focus, and calmness levels
- Performance metrics and improvement tracking
- Visual charts showing mood patterns
- Personalized recommendations

### Reports
- Downloadable reports with complete wellness history
- Activity breakdown and mood trends
- Game performance analysis

### User System
- Secure authentication with Supabase
- Cloud sync for registered users
- Guest mode for trying the app
- Profile management

## Tech Stack

- React 18 with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- React Router for navigation
- Recharts for data visualization
- Supabase for backend and authentication
- TanStack Query for state management
